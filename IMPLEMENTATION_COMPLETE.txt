KEY COMPONENTS EXPLAINED
================================================================================

GRID GENERATION
  • Creates 20×20 cell grid
  • Randomly places obstacles based on difficulty
  • Robot at (0, 0) - top-left
  • Goal at (19, 19) - bottom-right
  • Validates goal is reachable
  • Auto-regenerates if goal unreachable

A* PATHFINDING
  • Analyzes grid intelligently
  • Tracks g_score (cost from start)
  • Tracks h_score (heuristic distance to goal)
  • Tracks f_score (total estimated cost)
  • Always explores lowest f_score first
  • Records all explored nodes
  • Reconstructs optimal path

ROBOT ANIMATION
  • Follows A*-computed path
  • Moves one cell at a time
  • Speed controlled by user setting
  • Updates position in real-time
  • Displays as bright cyan dot

VISUALIZATION
  • Grid shown in center panel

  • Final path highlighted in yellow after pathfinding
  • Robot shown in bright cyan during animation
  • Goal shown in bright green
  • Obstacles shown in dark grey

TELEMETRY
