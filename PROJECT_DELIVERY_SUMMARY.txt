TROUBLESHOOTING QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════

"ModuleNotFoundError: No module named 'PySide6'"
→ Run: pip install pyside6

"No module named 'qt_material'"
→ Run: pip install qt-material
(Optional - app works without it)

"Application won't start"
→ Check Python version: python --version (need 3.7+)
→ Check dependency: python -c "from PySide6.QtWidgets import QApplication"

"GUI freezes during simulation"
→ This shouldn't happen! Contact support with system details

"Graphics look poor"
→ Update display drivers
→ Check display resolution
→ Try: pip install --upgrade pyside6
