                       UNDERSTANDING THE DISPLAY
================================================================================

GRID (Main area):
  Shows the 15×15 game board with:
  • Your robot (🤖) - moves along the path
  • Goal (🏁) - bottom-right corner
  • Walls (🧱) - obstacles (revealed by fog of war)
  • Paths (🟦) - walkable areas
  • Dots (·) - tiles the A* algorithm examined
  • Dark areas (⬛⬜) - hidden by fog of war

SIDEBAR (Right panel):
  Shows real-time information:
  • Position: Current X, Y coordinates (0-14)
  • Movement: Steps taken and path length
  • Stats: Nodes explored, efficiency, time
  • Progress: Percentage toward goal

================================================================================
                         KEY STATISTICS
================================================================================

NODES EXPLORED:
  What: How many cells the A* algorithm checked
  Meaning: More = more complex environment
  Goal: Varies by difficulty
  Easy: ~200-300 nodes
  Medium: ~400-600 nodes
  Hard: ~800-1200 nodes

EFFICIENCY:
  What: Path quality score (path length / nodes explored) × 100%
  Meaning: Higher is better (shorter path relative to search)
  Good: 50%+ (typical)
  Excellent: 75%+ (rare)
  Range: Usually 20-80%

TIME:
  What: Milliseconds to calculate the path
  Meaning: How fast the algorithm found the solution
  Typical: <1 millisecond (instant)
  Range: 0.1-2 milliseconds

PATH LENGTH:
  What: Number of steps to reach goal
  Meaning: Actual distance traveled
  Typical: 24-28 steps for 15×15 grid
  Varies by obstacles
