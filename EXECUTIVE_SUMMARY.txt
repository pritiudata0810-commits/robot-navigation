ALGORITHM DETAILS
================================================================================

A* PATHFINDING (Industry-Standard Optimal Algorithm)

Heuristic Function: Manhattan Distance
  Formula: |x1 - x2| + |y1 - y2|
  Optimal for: 4-directional movement (no diagonals)

Scoring System:
  • g_score: Actual cost from start node
  • h_score: Estimated cost to goal (heuristic)
  • f_score: Total cost (g_score + h_score)

Algorithm Process:
  1. Initialize start node with g=0, h=heuristic
  2. Add to open set (nodes to evaluate)
  3. While open set not empty:
     a. Select node with lowest f_score
     b. If goal reached, reconstruct and return path
     c. Expand neighbors and update scores
     d. Add new neighbors to open set
  4. If open set empty, no path exists
  5. Validate reachability; regenerate if needed

Result: Guaranteed shortest path from robot to goal
