# Robot Navigation AI Simulator - QUICKSTART

## 🚀 Get Started in 3 Steps

### Step 1: Install Dependencies
Open your terminal/command prompt and run:
```bash
pip install pyside6 qt-material pyqtgraph
```

### Step 2: Navigate to the Directory
```bash
cd D:\robot-navigation
```

### Step 3: Run the Simulator
```bash
python robot_navigation_simulator.py
```

That's it! A professional GUI window will open immediately.

---

## 📋 What You'll See

### Welcome Screen
- Large title: "Robot Navigation AI Simulator"
- Three buttons to choose from:
  - **Start Simulation** → Configure and run
  - **About Project** → Learn more
  - **Exit** → Close application

### Configuration Screen
Choose your settings:
- **Difficulty**: Easy (15%), Medium (30%), Hard (50%)
- **Speed**: Slow, Normal, or Fast animation

### Simulation Dashboard
Watch the A* algorithm in action:
- **Left panel**: Movement history log
- **Center panel**: Live 20×20 grid visualization
  - Green robot moving smoothly
  - Blue explored cells showing search progress
  - Green path highlighting optimal route
  - Dark cells as obstacles
- **Right panel**: Real-time statistics
- **Bottom**: Control buttons

### Analytics Screen
View final results:
- Total steps taken
- Path length
- Nodes explored
- Elapsed time
- Efficiency percentage

## 🐛 Troubleshooting

**"No module named 'PySide6'"**
```bash
pip install --upgrade pyside6
```

**"Application won't start"**
- Verify Python 3.7+ is installed: `python --version`
- Check dependencies: `pip list | grep -E "pyside|qt-material"`
- Try: `python robot_navigation_simulator.py --verbose`

**"GUI looks plain (no Material Design)"**
- qt-material is optional for styling
- Application still works, just less polished
- Install qt-material: `pip install qt-material`
